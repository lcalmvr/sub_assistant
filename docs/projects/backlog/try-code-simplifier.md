# Try Code Simplifier Plugin

**Priority:** Low
**Added:** 2025-01-24
**Type:** Tool to try

## What

Claude Code plugin that simplifies/refines code for clarity and maintainability.

## Install

```bash
claude plugin install code-simplifier
```

## Usage

After a long coding session or before merging a PR:
```
/code-simplifier
```

Or ask Claude to run the code-simplifier agent.

## When to Use

- End of long coding sessions
- Before merging complex PRs
- Periodic codebase cleanup
